font - size : 글꼴 크기 지정

            선택자 {
                font - size: 크기(px, em(배수));
            }
  • 고정크기 (px) 확인
  • 가변크기 (em) 확인
  • 가변크기 (%) 확인

font-weight : 글꼴 굵기 표현할 때 사용

            선택자{
                font-weight : nomal | bold | bolder | lighter | 100 ~ 900;
             }
  • 원래 글자
  • 굵게 표현
  • 원래 굵기보다 굵게 표현
  • 원래 굵기보다 가늘게 표현

font-variant : 영문 텍스트 문구를 작은 크기의 대문자로 변경

            선택자{
                font - variant : small - caps;
            }

font - style : 텍스트 문구를 기울여서 표현할 때 사용

            선택자{
                font-style: normal | italic | oblique;
            }

italic : 기울임 글꼴

oblique : 원래 글자를 기울여서 표현
