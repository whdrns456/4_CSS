텍스트 관련 스타일(text)

color : 텍스트 색상 지정

    선택자{
        color: 색상명 | 16진수 | rgb(x, x, x) | rgba(x,x,x,x)|hsl(x,x,x)

    }

text-decoration : 텍스트에 줄을 표시하거나 또는 표시 하지 않고자 할 때 사용

    선택자{
        text - decoration : none | underline | overline | line - through

    }
  • 텍스트에 줄 긋기 (상단)
  • 텍스트에 줄긋기 (중간)
  • 텍스트에 줄긋기 (하단)
네이버로

text - transform : 영문 텍스트의 대소문자 변환 시 사용 

        선택자 {
            text - transform : uppercase | lowercase | capitalize 
        }
  • uppercase : 모든 영문자를 대문자로 변환
  • Lowercase : 모든 영문자를 소문자로 변환
  • capitalize : 영문자의 첫번째 글자만 대문자로 변환

text -shadow : 텍스트에 그림자 효과를 줄 때 사용

        선택자{
            text-shadow: 가로거리x 세로거리y [번짐정도][색상]

        }
